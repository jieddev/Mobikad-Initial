# Undergraduate Capstone Project: "Mobikad: A Mobile-base Tricycle Booking System Using GPS Tracking and Monitoring System For The Local Government Unit Of Ormoc City"

The project aims to enhance efficiency, accessibility of tricycles, and safety for both passengers and drivers especially in rural cities for chartering tricycles.

Chartering vehicles especially in rural cities often consumes time, inaccessibility, and risking the safety of passengers and drivers.

An Android-based application for passengers and drivers and an Admin Web portal is developed using Flutter Framework, Firebase BaaS, and Cloud Functions. 
Integrated with multiple Google APIs through GGC for GPS vehicle tracking, Stripe for payment gateway, and Agora for in-app calling.
